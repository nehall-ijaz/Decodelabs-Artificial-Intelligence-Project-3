# 🤖 AI Recommendation System

A simple Artificial Intelligence Recommendation System developed as part of the DecodeLabs Artificial Intelligence Training Program.

## 📌 Project Overview

This project demonstrates the fundamental concepts of recommendation systems using similarity-based matching. The system collects user interests, compares them with available item features, calculates a similarity score, and recommends the most relevant courses or topics.

The project introduces core AI concepts such as:

* Content-Based Recommendation
* Similarity Matching
* Pattern Recognition
* User Preference Analysis
* Recommendation Logic

---

## 🎯 Objectives

* Accept user preferences as input.
* Analyze interests using similarity matching.
* Calculate recommendation scores.
* Display ranked recommendations.
* Understand the fundamentals of AI recommendation systems.

---

## 🛠 Technologies Used

* Python 3
* Lists
* Dictionaries
* Sets
* Sorting Algorithms

---

## ⚙️ How It Works

1. The user enters interests separated by commas.
2. The system compares user interests with item attributes.
3. A similarity score is calculated based on matching keywords.
4. Items are ranked according to their scores.
5. The most relevant recommendations are displayed.

## 📖 AI Concepts Implemented

This project uses a simple Content-Based Filtering approach where recommendations are generated by comparing user interests with predefined item features.

Similarity Score Formula:

Similarity Score = Number of Matching Interests

Higher scores indicate stronger recommendations.
